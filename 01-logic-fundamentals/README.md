# 🧩 Módulo: Fundamentos de Lógica em Java

Este módulo contém a base do meu aprendizado na linguagem Java, focado na resolução de problemas lógicos e matemáticos utilizando o ecossistema Maven.

## 🚀 Competências Desenvolvidas

### 📦 Processamento Sequencial
* Manipulação de variáveis e tipos primitivos.
* Entrada e saída de dados com a classe `Scanner`.
* Implementação de fórmulas matemáticas e conversão de dados.

### 🔀 Estruturas Condicionais
* **Controle de fluxo com `if-else`**: Implementação de decisões complexas baseadas em intervalos numéricos e lógica booleana.
* **Domínio de Lógica Aplicada**: Resolução de problemas reais como cálculos de Imposto de Renda, verificação de coordenadas cartesianas e validação de intervalos.
* **Conhecimento em `switch-case`**: Compreensão da sintaxe para simplificação de múltiplas condicionais de valor fixo, utilizada como alternativa ao `if-else` para melhoria da legibilidade em cenários específicos.

### 🔄 Estruturas de Repetição
* Domínio de laços `for` para iterações definidas.
* Uso de `while` para fluxos baseados em condições dinâmicas e validação de entradas.

## 📂 Organização de Pastas (Pacotes)

A estrutura segue o padrão de pacotes Java, organizada por categorias de lógica:

* `com.ulisses.sequential`: Exercícios de fluxo direto e cálculos básicos.
* `com.ulisses.conditionals`: Desafios de tomada de decisão com lógica booleana.
* `com.ulisses.repetition.forloop`: Laços com número definido de repetições.
* `com.ulisses.repetition.whileloop`: Laços baseados em condições dinâmicas.

*Este módulo serve como base sólida para o estudo de Programação Orientada a Objetos (POO).*
